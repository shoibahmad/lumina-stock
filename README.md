# Lumina Stock App

A beautiful, premium inventory management application built with Flutter.

## Features implemented
- **Product Listing:** Modern grid layout with premium card design.
- **UI/UX:** Light, airy color palette (Soft Blue/White) with Google Fonts (Poppins).
- **Data Model:** Prepared for Product Image, Name, Category, Buy/Sell Price.

## Upcoming Features (Planned)
- **Admin Panel:** Add, Edit, Delete products.
- **Cloudinary Integration:** Upload images directly to Cloudinary.
- **Authentication:** Admin vs User roles (User sees list, Admin manages inventory).

## How to Run

1.  Ensure you have Flutter installed.
2.  Run `flutter pub get` to install dependencies.
3.  Run `flutter run` to start the app on your emulator or device.

## Dependencies used
- `google_fonts`: For typography.
- `intl`: For currency formatting.
- `cloudinary_flutter`: For future image handling.
